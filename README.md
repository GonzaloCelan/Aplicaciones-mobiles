# App de Recordatorio de Medicación

Aplicación móvil desarrollada en React Native utilizando Expo.

La aplicación permite registrar medicamentos junto con un horario de toma, almacenarlos localmente en el dispositivo y recibir recordatorios mediante notificaciones locales.

---

# Funcionaliades

La app permite:

- Registrar medicamentos.
- Seleccionar horario de toma.
- Guardar la información localmente.
- Visualizar medicamentos registrados.
- Eliminar medicamentos.
- Recibir notificaciones locales como recordatorio.
